# SupplyChain Security Toolkit

Audit • Detect • Harden

A security audit toolkit focused on supply-chain attack surfaces:

- Linux privilege escalation & misconfigurations
- Drupal update & supply-chain security
- pfSense configuration audit
- CI/CD pipeline hardening

⚠️ This toolkit performs **non-destructive audits only**.  
No exploitation. No active attacks.

Built with Streamlit.
